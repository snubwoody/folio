
## Commit prefixes

feat: addition/updating of functionality
fix: bug correction
docs: addition/updating of documentation
style: modification of style and formatting of the code (spacings, commas, etc.)
refactor: modification of the sources being neither a fix nor an addition of functionality
test: addition or correction of test
chore: other updates not modifying either the sources or the tests
ci: modification of the configuration or scripts related to the CI (Travis, Circle, BrowserStack, SauceLabs, etc.)
revert: cancel (revert) a previous commit
